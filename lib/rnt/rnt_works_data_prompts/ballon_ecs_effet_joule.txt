### installation ballon d'ecs à effet joule (ballon_ecs_effet_joule) :

```jsx
{
norme_dispositif_regulation: enum(nf_en_iso_52120_1,autre), norme qui a été utilisée pour qualifier le dispositif de régulation répondant à la classe A;
sonde_de_mesure: boolean, présence d'une sonde qui mesure la température de stockage de l’eau dans le chauffe-eau - true, false;
service_analyse_donnes: boolean, présence d'un service d’analyse de données du chauffe-eau - true, false;
equipement_pilotage: boolean, présence d'un équipement de pilotage du chauffe-eau - true, false;
indication_surconsommation: boolean, présence d'un moyen de mesure ou d’estimation de la consommation d’eau chaude - true, false;
marque_systeme: string, nom de la marque du système;
reference_systeme: string, reference du système;
capacite_stockage_ballon: int, capacité de stockage du ballon d'eau chaude (unité : L);
}
```
