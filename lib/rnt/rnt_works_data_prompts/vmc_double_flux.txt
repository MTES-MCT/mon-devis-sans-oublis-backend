### installation vmc double flux :

```jsx
{
type_vmc_double_flux: enum(vmc_df_auto,vmc_df_hygro), type de VMC double flux (autoréglable, modulée hygroréglable);
type_installation: enum(installation_collective,installation_individuelle), type d'installation (collective ou individuelle);
nombre_logements: double, si installation collective : nombre de logements desservis (unité : sans unite);
classe_energetique_vmc: enum(a+,a,b,c,d,e,f,g), si installation individuelle : classe énergétique du système de ventilation selon le règlement européen (UE) n° 1254/2014;
efficacite_thermique_coll: double, si installation collective : efficacité thermique de l’échangeur de chaleur (unité : sans unite);
surface_habitable: double, si installation individuelle : surface habitable (unité : m²);
efficacite_thermique_ind: double, si installation individuelle : rapport de température de la centrale double flux (efficacité thermique) (unité : sans unite);
norme_rapport_temperature: enum(nf_en_13141_7,autre), norme qui a été utilisée pour la mesure du rapport de température de la centrale double flux;
norme_echangeur_chaleur: enum(nf_en_308,autre), norme qui a été utilisée pour la mesure de l'efficacité thermique de l'échangeur de chaleur;
certification_vmc: enum(nf_205,aahe,aare), certification (ind : NF 205, coll : AAHE ou AARE);
puissance_electrique_absorbee_caisson: double, si installation individuelle : puissance électrique absorbée pondérée du caisson (unité : WThC/(m3 /h));
avis_technique_cstb: string, si installation individuelle modulée hygroréglable : référence de l'avis technique CSTB;
date_avis_technique_cstb: date, si installation individuelle modulée hygroréglable : date de l'avis technique CSTB;
presence_systeme: boolean, présence d'un caisson, d'un échangeur de chaleur, de gaines, de bouches d’insufflation et d’extraction ou hygroréglables 0 : non 1 : oui;
marque_caisson: string, nom de la marque du caisson;
reference_caisson: string, reference du caisson;
marque_bouches_insufflation_extraction: string, nom de la marque des bouches d’insufflation et d’extraction;
reference_bouches_insufflation_extraction: string, référence des bouches d’insufflation et d’extraction;
}
```
